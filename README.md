# Variety Mission Hub

A fun web-based game inspired by variety shows, featuring secret missions and penalties for party games or social gatherings.

## Files

- `index.html`: The main game interface. This HTML file contains the interactive game where players can draw missions and penalties.

## How to Use

1. Open `index.html` in any modern web browser.
2. Click "New Mission" to randomly select a secret mission from categories like Food, Volleyball, Anime, Cafe, Social, and Universal, with varying difficulty levels (Easy, Med, Hard).
3. Click "Draw a Penalty" to reveal a random penalty (Batsu) such as drawing a mustache or eating a lemon.
4. Enjoy the game and remember the rule: If asked "Is that a mission?", you must tell the truth!

The game uses Tailwind CSS for styling and includes animations for a engaging experience.
