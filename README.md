# Variety Mission Hub

A fun web-based game inspired by variety shows, featuring secret missions and penalties for party games or social gatherings.

## Files

- `index.html`: The main game interface. This HTML file contains the interactive game where players can draw missions and penalties.
- `deck.csv`: A CSV file containing the full deck of missions. It includes categories such as Universal, Food & Cafe, Volleyball, Anime, Physical, Conversation, Skill, and Party, with mission descriptions and difficulty levels (Easy, Medium, Hard).
- `variety_night.jpg`: An image file, likely used for visual elements in the game or documentation.
- `README.md`: This file, providing project information and usage instructions.

## How to Use

1. Open `index.html` in any modern web browser.
2. Click "New Mission" to randomly select a secret mission from various categories with varying difficulty levels (Easy, Medium, Hard).
3. Click "Draw a Penalty" to reveal a random penalty (Batsu) such as drawing a mustache or eating a lemon.
4. Enjoy the game and remember the rule: If asked "Is that a mission?", you must tell the truth!

The game uses Tailwind CSS for styling and includes animations for an engaging experience. The mission deck is sourced from `deck.csv` for a wide variety of challenges.
